As playlists são uma ótima maneira de criar coleções de músicas, seja para só você ouvir ou para compartilhar. Para criar uma:

1. Clique na “Sua Biblioteca”.

2. Clique em CRIAR.

3. Dê um nome para a playlist.

4. Comece a incluir algumas músicas (enquanto recomendamos outras).
